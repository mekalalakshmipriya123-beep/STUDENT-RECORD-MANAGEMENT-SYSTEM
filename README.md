# STUDENT-RECORD-MANAGEMENT-SYSTEM
A modular C-based Student Record Management System using Singly Linked Lists and file handling for efficient record management.

## Features
- Add new student records
- Delete records
- Modify records
- Display records
- Sort by Name or Percentage
- Save and Load records using `student.dat`
- Automatic roll number assignment

## Technologies Used
- C Programming
- Singly Linked List
- Dynamic Memory Allocation
- File Handling
- Modular Programming
